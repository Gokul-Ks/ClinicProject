
Auth APIS ==================================================================================

In case of error:

Response: 
{
  success: false,
  error: "Error Message"
}






/auth/login - POST

Body Expected:
{
    "email": "example@gamil.com",
    "password": "password"
}

Output: If Success
{
    success: true,
    message: "Login Successful",
    profile: {
      name: "John Doe",
      email: "example@gamil"
      role: "",
    }
}
